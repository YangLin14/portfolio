As a Software Development intern, I developed the OrderEase mobile application. This project was driven by a personal mission to empower individuals navigating English as a second language, helping them order food with confidence and ease.

Project Goal & Motivation

As an immigrant from Taiwan, I experienced the anxiety of ordering food in a non-native language. This discomfort motivated me to develop a mobile application aimed at empowering individuals in similar situations. My goal was to create an innovative solution that provided real-time language translation and used simple, accurate images to convey essential information quickly and reduce social friction.

Core Features

The application was developed using Flutter to provide a seamless cross-platform experience. Key features included:

  • Restaurant Discovery: Efficiently search for nearby restaurants.
  • Visual Menus: Choose meals with clear images and recipe descriptions.
  • Sentence Generation: Automatically generate grammatically correct ordering sentences.
  • Audio Assistance: An integrated audio service helps convey preferences for flavor and dietary requirements.
  • Special Order Advice: Offers guidance on how to place custom or special orders.
