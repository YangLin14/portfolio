ListVaults 🎬

The ultimate tool for organizing and tracking your video library, from movies and TV shows to web series.

Project Summary 📝

ListVaults is a tool for organizing and tracking your video library. It allows you to create personalized lists with details like genre, episodes, priority, and notes. Each list is divided into "To-Watch" and "Watched" sections, making it easy to manage your viewing queue. With secure cloud storage powered by Firebase, your lists are always safe and accessible.

Features ✨

Core Functionality ⚙️

  • 📝 Create personalized lists for any video type.
  • ✅ Separate "To-Watch" and "Watched" sections.
  • ➕ Add custom details: genre, episodes, and priority.
  • ☁️ Secure cloud storage via Firebase.

User Experience 🎨

  • 🚀 Simple sign-up process.
  • ↔️ Easily move items between lists.
  • ✏️ Fully editable entries at any time.
  • ✨ Clean interface for efficient management.
